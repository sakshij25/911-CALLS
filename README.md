# 911-CALLS

## Description
 This project is regarding analysis of emergency calls.
  1. Monthly calls
  2. Daily calls 
  3. Hourly calls
    
## Language used -
 Python

## Library Used -
 Pandas (to do all the analysis).
 Matplotlib and Seaborn (to do all the visualization)
